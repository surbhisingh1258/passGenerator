# 🔐 Password Generator

A responsive and customizable password generator built using JavaScript, React, HTML, and CSS. This project allows users to create secure passwords based on their preferences, ensuring both usability and strong security practices.

## 🚀 Features
- 🔢 Adjustable password length
- 🔐 Option to include:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- 📋 One-click copy to clipboard
- 📱 Fully responsive design for all devices
- ⚡ Fast and user-friendly interface
- 
## 🛠️ Tech Stack
- Frontend: React, JavaScript
- Styling: HTML, CSS
